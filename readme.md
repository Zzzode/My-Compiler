## Introduction

本项目为编译原理课程项目，目的是完成一个C语言编译器。

## Environment

- 系统：win10
- IDE：Jerbrain Clion
- 编译器：MinGW_w64
- 工作语言：C++
- 目标语言：C-
- 实现目标语言编译器：C-
- 编译器目标语言：x86汇编(有改动，不完整) + 自己实现的x86虚拟机(不完整)
